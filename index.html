Algorithms & Visualization Project 6

This is a simple visualization of Graham's scan algorithm, which is used compute the convex hull of a set of points. The instructions required to run the algorithm are explained below:

  • The user can click on the blank canvas below to add as many points as desired.
  • Clicking "START" will begin the demo and allow the user to click "STEP" and "ANIMATE".
  • Clicking "STEP" illustrates a single stage of the algorithm.
  • Clicking "ANIMATE" illustrates each stage of the algorithm at regular time intervals until the algorithm terminates.
